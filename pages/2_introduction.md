# Builder Pattern - Definition

Das Builder Pattern ist ein **Creational Design Pattern**, das die Konstruktion komplexer Objekte von ihrer Repräsentation trennt.

<v-clicks>

## Hauptziele

- **Schritt-für-Schritt-Konstruktion** komplexer Objekte
- **Gleicher Konstruktionsprozess** kann verschiedene Repräsentationen erzeugen
- **Trennung** von Konstruktionslogik und Objektrepräsentation
- **Fluent Interface** für bessere Lesbarkeit

</v-clicks>
